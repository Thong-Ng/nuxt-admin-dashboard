## Getting Started

### Prerequisites

- Node.js
- npm / yarn / pnpm

### Setup

```bash
# Clone the repository
git clone https://github.com/your-username/nuxt-admin-dashboard.git

# backend setup
cd nuxt-admin-dashboard/backend
npm install
#add .env
npm start dev

# frontend setup
cd nuxt-admin-dashboard/frontend-admin
npm install
npm run dev
```

### Build for Production

```bash
npm run build
npm run start
```
